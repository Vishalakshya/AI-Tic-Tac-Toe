# AI-Tic-Tac-Toe
An AI based Tic-Tac-Toe game with modern GUI

A modern GUI-based Tic Tac Toe game built using **Python** and **Tkinter** with an unbeatable AI powered by the **Minimax Algorithm with Alpha-Beta Pruning concepts**. The project features an interactive user interface, intelligent gameplay, and smooth game experience. 

---

## 🚀 Features

* Modern and responsive GUI using Tkinter
* Human vs AI gameplay
* Intelligent AI using Minimax Algorithm
* Winning line highlighting
* Draw detection
* Reset game functionality
* Smooth UI animations and colors
* Unbeatable AI opponent

---

## 🛠️ Tech Stack

* **Python**
* **Tkinter**
* **Minimax Algorithm**
* **AI Game Logic**

---

## 🧠 How the AI Works

The AI uses the **Minimax Algorithm** to evaluate all possible game outcomes and choose the best move. The algorithm recursively simulates future game states and selects the optimal move to maximize AI victory chances while minimizing player victory chances.

The implementation ensures:

* Best possible move selection
* Draw optimization
* Impossible-to-beat gameplay

## 🎯 Game Rules

* Player uses **X**
* AI uses **O**
* First player to align 3 symbols horizontally, vertically, or diagonally wins
* If all cells are filled without a winner, the game ends in a draw
